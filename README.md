# VOICEVOX TELEMETRY with GAS

VOICEVOX の辞書を収集する Google Apps Script コード

## ビルド

```
npm run build
```

## デプロイ

```
npm run push
npm run deploy
```

デプロイするには、[clasp](https://www.npmjs.com/package/@google/clasp) にログインしておく必要があります。

また、
```
npm start
# または npm run start
```
で、ビルド→デプロイをすることができます。
